🏟️ Venue System Test

📋 Test Instructions

This page helps you test if the venue system is working properly. Follow these steps:

  1. Make sure your React app is running
  2. Check if you can access the venues page
  3. Test the admin venue management
  4. Verify database connection

🔗 Navigation Tests

Test these links in your app:

  • Venues Page: /venues
  • Admin Dashboard: /admin (then click Venues tab)
  • Homepage: Check if featured venues section appears

🗄️ Database Tests

Check if the database migration has been run:

  • Go to Supabase Dashboard
  • Check if sports_facilities table exists
  • Verify sample venues are present

🐛 Common Issues & Fixes

Issue: "Cannot read property 'map' of undefined"

Fix: Check if NEPAL_PROVINCES, SPORTS_TYPES, AMENITIES are properly imported

Issue: "venueDiscoveryService.getById is not a function"

Fix: Make sure getById is exported in venueDiscoveryService

Issue: "Table 'sports_facilities' doesn't exist"

Fix: Run the database migration in Supabase
